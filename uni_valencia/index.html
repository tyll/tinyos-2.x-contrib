Currently available projects:

EHarvester

EHarvester is a data gathering application based on previous Harvester (see ETHZ contribs) designed for enviromental monitoring purpose. It takes samples (T and RH) every 10 minutes and sent ir to the sink using ctp routing protocol and synchronous LPL. It is designed for TelosB mote and Tinyos-2.1.0.

Gatherer

Gatherer is a data gathering application designed over IPv6 using BLIP-1.0, for enviromental monitoring purpose. It takes samples (T and RH) and saves locally, waiting till the sink asks for it. It is designed for TelosB mote and Tinyos-2.1.1.
